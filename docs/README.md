## Szybki start

1. Sklonuj repozytorium
2. Zainstaluj zależności: `npm install`
3. Skonfiguruj bazę danych: `npx prisma db push`
4. Zasilij danymi testowymi: `npx prisma db seed`
5. Uruchom aplikację: `npm run dev`
