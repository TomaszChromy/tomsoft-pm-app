# Screenshoty TomSoft PM

## Instrukcja dodawania screenshotów

1. Zrób screenshot aplikacji w rozdzielczości 1920x1080
2. Zapisz w formacie PNG
3. Nazwij pliki zgodnie z konwencją:
   - `dashboard.png` - Główny dashboard
   - `projects.png` - Lista projektów
   - `project-detail.png` - Szczegóły projektu
   - `tasks-kanban.png` - Tablica Kanban
   - `task-detail.png` - Szczegóły zadania
   - `team.png` - Zarządzanie zespołem
   - `analytics.png` - Dashboard analityczny
   - `login.png` - Strona logowania
   - `mobile-dashboard.png` - Wersja mobilna

## Wymagania techniczne

- **Format**: PNG
- **Rozdzielczość**: 1920x1080 (desktop), 375x812 (mobile)
- **Jakość**: Wysoka (bez kompresji)
- **Rozmiar**: Max 2MB na plik
