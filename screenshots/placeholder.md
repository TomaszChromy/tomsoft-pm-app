## Narzędzia do screenshotów

### Windows
- **Snipping Tool** (wbudowane)
- **Greenshot** (darmowe)
- **LightShot** (darmowe)

### macOS
- **Cmd + Shift + 4** (wbudowane)
- **CleanShot X** (płatne)

### Linux
- **GNOME Screenshot** (wbudowane)
- **Flameshot** (darmowe)

## Wymagania techniczne

- **Format**: PNG
- **Rozdzielczość**: 1920x1080 (desktop)
- **Jakość**: Wysoka, bez kompresji
- **Rozmiar**: Max 2MB na plik
- **Nazwy plików**: Zgodnie z konwencją powyżej

## Po dodaniu screenshotów

1. Usuń ten plik placeholder.md
2. Zaktualizuj README.md jeśli potrzeba
3. Dodaj pliki do git:
   ```bash
   git add screenshots/
   git commit -m "Add application screenshots"
   ```
